# Build-an-Item-Catalog-Application-Udacity-Fullstack
Udacity Full Stack Web Developer Nanodegree Project

## Requirements
python3
flask 0.12.2
flask-login 0.4.0
flask_sqlalchemy

## How to run
```bash
export FLASK_APP=app.py
flask run

* Running on http://127.0.0.1:5000/
```
